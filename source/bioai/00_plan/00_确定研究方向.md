# 目前世界上对基因🧬研究的最新进展是什么？

> ChatGPT 2025-10-20

好——给你一份**截至最近（2025 年 10 月下旬）**整理的“全球基因研究最新进展速览”，按主题把最重要、最具临床/技术影响力的进展列出来，并说明为什么重要和可能的应用/限制。下面每条都带有来源，方便你继续深挖。

### 1) **首例人体 Prime-editing（“精确修复”）临床数据**

- 进展：2025 年春季有团队（Prime Medicine 等）公布了首例使用 **prime editing** 治疗的临床数据，显示在首位受试者中出现了安全性与疗效的初步证据（用于某些罕见免疫/血液疾病）。这是 prime editing 第一次在人体层面获得正面临床信号。[CRISPR Medicine+1](https://crisprmedicinenews.com/news/first-ever-prime-editing-therapy-shows-safety-and-efficacy-in-patient-with-chronic-granulomatous-dis/?utm_source=> ChatGPT.com)
- 意义：prime editing 不产生双链断裂，理论上更安全，能更精确地修复单碱基或短片段突变。若继续验证成功，会成为纠正遗传病的重要新工具。
- 限制：仍属早期临床，长期安全性、有效率与大规模可及性需进一步验证。[PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC12322275/?utm_source=> ChatGPT.com)

## 2) **体内（in-vivo）基因编辑与 LNP（脂质纳米颗粒）递送取得突破**

- 进展：儿童医院 / Penn 等机构实现了**针对个体的体内 CRISPR 治疗**（用 LNP 将编辑物质递送至患者器官），并报道单例成功与良好短期安全性；同时多项论文展示了改良 LNP 能把编辑工具送到肺、肝等体外以外的器官。[chop.edu+2mitchell-lab.seas.upenn.edu+2](https://www.chop.edu/news/worlds-first-patient-treated-personalized-crispr-gene-editing-therapy-childrens-hospital?utm_source=> ChatGPT.com)
- 意义：若体内递送普遍可行，就能避免“体外改造干细胞再回输”这种复杂流程，扩大应用到更多器官与更常见疾病（如心血管、代谢病、遗传性器官病等）。
- 限制：器官靶向、免疫反应、脱靶编辑和剂量安全窗口仍是关键挑战。[mitchell-lab.seas.upenn.edu](https://mitchell-lab.seas.upenn.edu/wp-content/uploads/2025/04/Haley_ACSNano.pdf?utm_source=> ChatGPT.com)

## 3) **已有基因编辑/基因疗法开始进入临床常规与监管许可（转化成果）**

- 进展：例如 exagamglogene autotemcel（商品名 CASGEVY / exa-cel）与 Casgevy 等基因编辑/细胞疗法已获得 FDA 批准并被一些国家/地区纳入临床使用或医保决策（用于镰状细胞/β 地中海贫血等）。英国 NHS 也批准了相关高价基因治疗的有限使用。[U.S. Food and Drug Administration+1](https://www.fda.gov/vaccines-blood-biologics/casgevy?utm_source=> ChatGPT.com)
- 意义：表明从实验室到患者的转化正在加速，且监管机构已开始接受编辑型细胞疗法的风险/收益框架。
- 限制：价格高、适应证窄、长期疗效/安全需随访监测。[Ash Publications](https://ashpublications.org/ashclinicalnews/news/7796/Gene-Editing-Therapy-Receives-Second-FDA-Approval?utm_source=> ChatGPT.com)

## 4) **基因编辑工具本身的技术迭代（效率与范围显著提升）**

- 进展：学术界/产业界都在优化 prime editing（如 proPE 等改良版）和新颖系统（例如最近报道的 retron-based 编辑系统），这些工作大幅提高某些难改位点的编辑效率并扩展可编辑碱基种类。[Nature+1](https://www.nature.com/articles/s41929-025-01406-6?utm_source=> ChatGPT.com)
- 意义：工具越强、越灵活，就能覆盖更多的人类致病变异，使更多遗传病成为潜在治疗目标。
- 限制：更高效的工具也需要更严密的脱靶评估与安全验证。[Nature](https://www.nature.com/articles/s41929-025-01406-6?utm_source=> ChatGPT.com)

## 5) **“N-of-1”（个体化/单例）快速定制基因疗法与法规伦理讨论**

- 进展：有高影响力期刊报道团队为超罕见或唯一病例快速设计、制造并在体内/体外实施个体化基因编辑（N-of-1 病例），并发表初步疗效/安全数据。与此同时，围绕伦理、监管和制造可重复性展开讨论。[nejm.org+1](https://www.nejm.org/doi/full/10.1056/NEJMoa2504747?utm_source=> ChatGPT.com)
- 意义：为极罕见病患者带来希望，也推动监管机构思考如何在保证安全的同时允许快速个体化干预。
- 限制：可扩展性差、成本高、伦理/知情同意/长期跟踪是难题。[nejm.org](https://www.nejm.org/doi/full/10.1056/NEJMoa2504747?utm_source=> ChatGPT.com)

## 6) **AI 与大数据加速基因组学（设计、解读、靶点发现）**

- 进展：AI／大模型已经被用于加速 CRISPR 载体设计、脱靶预测、蛋白结构预测和表型预测（近年有多起学术/产业新闻，显示 AI 在实验设计和缩短开发周期方面的实际应用）。近期有团队探索专门用于基因编辑设计的“大模型”（如“CRISPR-GPT”类研究）来提高效率。[Stanford Medicine](https://med.stanford.edu/news/all-news/2025/09/ai-crispr-gene-therapy.html?utm_source=> ChatGPT.com)
- 意义：能降低研发成本和时间、提高命中率，使更多候选靶点和编辑策略能够被快速筛选。
- 限制：模型偏见、训练数据限制、可解释性和法规可接受性仍需解决。[Stanford Medicine](https://med.stanford.edu/news/all-news/2025/09/ai-crispr-gene-therapy.html?utm_source=> ChatGPT.com)

------

## 总结（一句话）

2024–2025 年间，基因编辑从“技术演示”向“临床可用”快速迈进：**prime editing 首次有人体临床数据、体内 LNP 递送的个体化 CRISPR 已开始在人类试验中出现、多个编辑型细胞疗法获得监管批准或纳入临床使用**，同时编辑工具（proPE、retron 等）与 AI 设计正提升效率与可及性——但长期安全、脱靶、递送与成本仍是必须解决的瓶颈。[nejm.org+4CRISPR Medicine+4chop.edu+4](https://crisprmedicinenews.com/news/first-ever-prime-editing-therapy-shows-safety-and-efficacy-in-patient-with-chronic-granulomatous-dis/?utm_source=> ChatGPT.com)

# 如何加入AI 与大数据加速基因组学的研究？

>  > ChatGPT 2025-10-20

这是一个非常棒的问题——你问的其实正是现在基因科学里最具前景的交叉领域之一：**“AI + 大数据驱动的基因组学”**（AI-driven Genomics）。
 下面我给你详细梳理一条**实际可行的路线图**：包括所需背景、具体研究方向、可加入的机构与项目、技术栈建议，以及如何入门。

------

## 🧩 一、理解你要进入的“交叉点”

AI 与大数据在基因研究中的核心作用是三大类：

| 方向                                 | 核心任务                                                     | 举例                                     |
| ------------------------------------ | ------------------------------------------------------------ | ---------------------------------------- |
| **1. 基因组数据分析**                | 用机器学习/深度学习解析海量基因组序列、突变数据、表观组、转录组 | variant calling、基因表达预测、GWAS 分析 |
| **2. AI 辅助功能预测与药物靶点发现** | 预测蛋白结构、突变功能影响、非编码区调控作用                 | AlphaFold、DeepMind Enformer、Basenji    |
| **3. AI 驱动的基因编辑设计与模拟**   | 用生成模型设计 guide RNA、预测脱靶、优化编辑系统             | CRISPR-GPT、DeepCRISPR、PrimeDesign      |

------

## 🧠 二、所需核心能力（按重要度排序）

| 技能领域                          | 说明                                      | 推荐工具/语言                                           |
| --------------------------------- | ----------------------------------------- | ------------------------------------------------------- |
| **1. 计算生物学基础**             | 基因组学、转录组、表观遗传学知识          | 熟悉 FASTQ、BAM、VCF 文件；Ensembl、UCSC Genome Browser |
| **2. Python/机器学习**            | 模型训练、特征提取、神经网络设计          | Python, PyTorch, TensorFlow, scikit-learn               |
| **3. 数据处理与大数据分析**       | 处理 TB 级别数据集                        | Linux, Bash, Snakemake, Nextflow, Spark                 |
| **4. 生物信息工具链**             | 分析管线与常用软件                        | samtools, GATK, STAR, bedtools, Bioconductor            |
| **5. AI 模型与结构生物学结合**    | 利用深度学习预测蛋白或序列功能            | AlphaFold, ESM-2, DeepVariant, Enformer                 |
| **6. 英文学术写作与代码复现能力** | 能复现 GitHub 项目、读懂 bioRxiv preprint | GitHub, Papers with Code, Colab                         |

------

## 🔬 三、研究方向（适合新加入者的切入点）

1. **AI 辅助基因突变功能预测**
   - 用 CNN 或 Transformer 模型预测 SNP/突变的功能影响（如DeepSEA、Enformer的思路）。
2. **CRISPR 编辑靶点优化模型**
   - 结合 AI 预测脱靶概率、guide RNA 效率。
3. **蛋白结构与功能预测**
   - 参与 AlphaFold/ESM 生态的改进或应用。
4. **AI for Multi-omics 数据整合**
   - 把基因组、转录组、表观组、代谢组数据融合建模（multi-modal deep learning）。
5. **群体基因组学 + AI 模式发现**
   - 用无监督学习在大规模人群测序数据中找出潜在疾病相关模式。
